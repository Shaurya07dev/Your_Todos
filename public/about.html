About This Project

This is a simple Todolist app built with Express.js, Supabase (for database and Google authentication), HTML, and Tailwind CSS. The app allows users to manage their todos and supports Google login. Admin users can manage all users and todos.

Tech Stack

  • Express.js (Backend)
  • Supabase (Database & Auth)
  • HTML & Tailwind CSS (Frontend)
  • Vercel (Hosting)
